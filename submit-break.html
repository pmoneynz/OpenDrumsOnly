Submit a new drum break

Found a break that should be in the catalog? Send it with the form below. This opens a prefilled GitHub issue so submissions stay transparent and trackable.

Submission guidelines

  • Include at least artist + track title so the team can verify.
  • Discogs release URLs significantly speed up review.
  • If possible, mention where the break sits in the track.
  • Duplicate submissions are fine; they help confirm demand.
